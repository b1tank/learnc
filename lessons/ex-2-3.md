> **Exercise 2-3.** Write a function `htoi(s)` that converts a string of hexadecimal digits — optionally prefixed with `0x` or `0X` — to its integer equivalent. The allowed digits are `0`-`9`, `a`-`f`, and `A`-`F`.

## Approach

Walk the string from left to right, accumulating the value: `result = result * 16 + digit_value`. For each character, map it to its 0-15 digit value:

- `'0'`–`'9'` → `c - '0'`
- `'a'`–`'f'` → `c - 'a' + 10`
- `'A'`–`'F'` → `c - 'A' + 10`

```c:starter
#include <stdio.h>
#include <ctype.h>

int htoi(const char s[]);

int main(void) {
    printf("%d\n", htoi("0xFF"));       /* 255 */
    printf("%d\n", htoi("0X10"));       /* 16 */
    printf("%d\n", htoi("1aB"));        /* 427 */
    printf("%d\n", htoi("0"));          /* 0 */
    return 0;
}

int htoi(const char s[]) {
    /* TODO */
    return 0;
}
```

## Solution

```c
#include <ctype.h>

static int hex_digit(int c) {
    if (c >= '0' && c <= '9') return c - '0';
    if (c >= 'a' && c <= 'f') return c - 'a' + 10;
    if (c >= 'A' && c <= 'F') return c - 'A' + 10;
    return -1;     /* not a hex digit */
}

int htoi(const char s[]) {
    int i = 0;
    int value = 0;

    /* skip optional 0x / 0X prefix */
    if (s[0] == '0' && (s[1] == 'x' || s[1] == 'X'))
        i = 2;

    for (; s[i] != '\0'; ++i) {
        int d = hex_digit(s[i]);
        if (d < 0) break;          /* stop at first non-hex character */
        value = value * 16 + d;
    }
    return value;
}
```

```output
255
16
427
0
```

## Notes from the author

- The hex-digit helper is the heart of the function. Pulling it out as a separate function (or a small `switch`) keeps the main loop readable and lets you unit-test the mapping in isolation.
- I made the function tolerant: on an invalid character it stops and returns what it has so far. The K&R-era convention is "trust the caller to pass valid input"; the modern convention is "return an error indicator or a status code." For a teaching exercise the lenient version is fine; for production input parsing, return `int` and write the converted value through a pointer parameter, returning the number of digits consumed (the strtoul convention).
- `<ctype.h>` has `isxdigit(c)` that tests "is this a hex digit?" and you can use the standard `tolower(c)` to fold case before the digit conversion. Cleaner, fewer branches.
