> **Exercise 2-10.** Rewrite a function that converts uppercase letters to lowercase, using a conditional expression in place of the `if`-statement.

## Approach

The condition is simple: "if this char is between `'A'` and `'Z'`, add `'a' - 'A'` to it; otherwise leave it." That's a textbook ternary.

```c:starter
#include <stdio.h>
#include <string.h>

int lower(int c);

int main(void) {
    char s[] = "Hello, World!";
    for (int i = 0; s[i] != '\0'; ++i)
        s[i] = lower(s[i]);
    printf("%s\n", s);
    return 0;
}
```

## Solution

```c
int lower(int c) {
    return (c >= 'A' && c <= 'Z') ? c + ('a' - 'A') : c;
}
```

```output
hello, world!
```

## Notes from the author

- The literal value of `'a' - 'A'` is 32 on ASCII. Writing it as `('a' - 'A')` instead of hard-coding `32` documents the intent and survives a hypothetical port to non-ASCII (where the offset might be different).
- The ternary form makes `lower` a single-line expression — it slots into a `return`, an initialiser, or even a macro without restructuring. That's the whole point of `?:` over `if`/`else`: it's *composable* into expressions.
- The standard `tolower(c)` in `<ctype.h>` does the same thing, but is locale-aware (in some locales additional characters have case mappings). For pure ASCII processing the exercise version is fine; for human text, prefer the library function.
