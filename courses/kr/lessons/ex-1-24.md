> **Exercise 1-24.** Write a program to check a C program for rudimentary syntax errors like unbalanced parentheses, brackets, and braces. Don't forget about quotes — both single and double — and comments. If you want to be ambitious, handle escape sequences inside strings.

## Approach

Two pieces compose:

1. **A stripped/skipped version of the input.** Strings, character constants, and comments don't count toward balance. So we reuse the state machine from ex-1-23 — but instead of *emitting* code, we *flag* opening/closing delimiters when we're in `CODE` state.
2. **A stack of opening delimiters.** Each opener pushes; each closer pops and must match (e.g., `)` must match `(`). At EOF the stack must be empty.

```c:starter
#include <stdio.h>

#define MAXSTACK 1000

int main(void) {
    /* TODO: balance checker */
    return 0;
}
```

## Solution

```c
#include <stdio.h>

#define MAXSTACK 1000

enum state { CODE, STRING, CHAR_LIT, LINE_CMT, BLOCK_CMT };

static char stack[MAXSTACK];
static int  sp = 0;
static int  line = 1;
static int  errors = 0;

static char matching_open(char close) {
    switch (close) {
        case ')': return '(';
        case ']': return '[';
        case '}': return '{';
    }
    return 0;
}

static void report(const char *msg, int ln) {
    fprintf(stderr, "line %d: %s\n", ln, msg);
    ++errors;
}

static void handle_open(char c) {
    if (sp < MAXSTACK) stack[sp++] = c;
    else report("delimiter stack overflow", line);
}

static void handle_close(char c) {
    char want = matching_open(c);
    if (sp == 0) {
        char msg[64];
        snprintf(msg, sizeof msg, "stray '%c' with no matching opener", c);
        report(msg, line);
    } else if (stack[sp - 1] != want) {
        char msg[80];
        snprintf(msg, sizeof msg, "'%c' does not match opener '%c'", c, stack[sp - 1]);
        report(msg, line);
        --sp; /* recover by popping anyway */
    } else {
        --sp;
    }
}

int main(void) {
    int c, pending = 0;
    enum state st = CODE;

    while ((c = getchar()) != EOF) {
        if (c == '\n') ++line;

        if (st == CODE) {
            if (pending) {
                pending = 0;
                if (c == '*') { st = BLOCK_CMT; continue; }
                if (c == '/') { st = LINE_CMT;  continue; }
                /* otherwise the prior '/' was just a divide op — no action */
            }
            switch (c) {
                case '/':  pending = 1; break;
                case '"':  st = STRING;   break;
                case '\'': st = CHAR_LIT; break;
                case '(': case '[': case '{':
                    handle_open((char)c); break;
                case ')': case ']': case '}':
                    handle_close((char)c); break;
            }
        } else if (st == STRING || st == CHAR_LIT) {
            if (c == '\\') {
                int n = getchar();         /* skip escaped char */
                if (n == '\n') ++line;
                if (n == EOF) break;
            } else if ((st == STRING && c == '"') ||
                       (st == CHAR_LIT && c == '\'')) {
                st = CODE;
            } else if (c == '\n') {
                report("unterminated string/char constant", line - 1);
                st = CODE;                 /* recover */
            }
        } else if (st == LINE_CMT) {
            if (c == '\n') st = CODE;
        } else if (st == BLOCK_CMT) {
            if (c == '*') {
                int n = getchar();
                if (n == '/') st = CODE;
                else if (n == '\n') ++line;
                else if (n == EOF) break;
            }
        }
    }

    /* anything left on the stack is unclosed */
    while (sp > 0) {
        char msg[64];
        snprintf(msg, sizeof msg, "unclosed '%c'", stack[--sp]);
        report(msg, line);
    }
    if (st == BLOCK_CMT) report("unterminated block comment", line);

    if (errors == 0) {
        printf("OK\n");
        return 0;
    }
    fprintf(stderr, "%d problem%s found\n", errors, errors == 1 ? "" : "s");
    return 1;
}
```

```output
OK
```

Try corrupting the program: delete a `}` and the checker reports `line N: unclosed '{'`. Add a stray `]` and it reports `line N: stray ']' with no matching opener`.

## Notes from the author

- This is the first program in the book that actually *parses*. The architecture — lexer state machine feeding tokens to a higher-level analyser — is exactly how every compiler, linter, and code formatter is built. You're holding the seed of a real tool.
- The stack is just a `char[]` plus a stack pointer. That's the most fundamental data structure in computing; we'll see it again in chapters 4 (recursion) and 6 (linked structures).
- I added line tracking and error messages because a checker without locality info is useless on real programs. Notice how the `\n` increments and the `report(msg, line)` calls thread through every state.
- **What this misses** (deliberately, in K&R scope): preprocessor directives (`#if` / `#endif` balance), template angle brackets (not in C), trigraphs/digraphs, raw string literals, the `<...>` of `#include`. Real-world linters add all of these.
