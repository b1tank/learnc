> **Exercise 2-4.** Write a variant of `squeeze(s1, s2)` that removes from `s1` every character that occurs anywhere in the string `s2`.

## Approach

Walk `s1` with a read index `i` and a write index `j`. For each character `s1[i]`, check whether it appears in `s2`. If not, copy it to `s1[j]` and advance `j`. The compaction is in-place.

The "appears in s2" test is a small inner loop - for short alphabets, $O(|s2|)$ is fine. For larger alphabets a 256-element boolean table would make it $O(1)$.

```c:starter
#include <stdio.h>

void squeeze(char s1[], const char s2[]);

int main(void) {
    char buf[] = "hello, world";
    squeeze(buf, "lo");
    printf("%s\n", buf);    /* expect: he, wr */
    return 0;
}

}
```

## Solution

```c
#include <stdio.h>

static int in_string(int c, const char s[]) {
    for (int k = 0; s[k] != '\0'; ++k)
        if (s[k] == c) return 1;
    return 0;
}

void squeeze(char s1[], const char s2[]) {
    int j = 0;
    for (int i = 0; s1[i] != '\0'; ++i) {
        if (!in_string(s1[i], s2)) {
            s1[j++] = s1[i];
        }
    }
    s1[j] = '\0';   /* re-terminate */
}

int main(void) {
    char buf[] = "hello, world";
    squeeze(buf, "lo");
    printf("%s\n", buf);
    return 0;
}
```

```output
he, wr
```

## Notes from the author

- The two-index scan is the canonical **in-place compaction** pattern. You'll see it again in array filtering, deduplication, and stream parsers. Internalise it: read pointer always ≥ write pointer; copy survives, drop is implicit.
- The terminating `s1[j] = '\0'` is **non-optional**. Without it, characters past the new logical end remain in the buffer - old bytes that are now garbage from the caller's perspective. Forgetting the terminator is one of the classic C bugs in this kind of routine.
- For very short `s2` (the common case), the linear `in_string` is fine. For "delete every digit" or "delete all whitespace," replace it with `isdigit(c)` / `isspace(c)` from `<ctype.h>`. For "delete any of these 256 bytes," precompute a `char drop[256]` lookup table once and replace the call with `drop[(unsigned char)c]`.
