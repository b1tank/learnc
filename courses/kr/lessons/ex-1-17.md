> **Exercise 1-17.** Write a program that prints every input line that is longer than 80 characters.

## Approach

Reuse the `read_line` helper from §1.9 (improved in ex-1-16 so it reports the *real* length). If `len > 80`, print the line.

```c:starter
#include <stdio.h>

#define MAXLINE 1000
#define THRESHOLD 80

int read_line(char line[], int maxsize);

int main(void) {
    /* TODO: print lines longer than THRESHOLD */
    return 0;
}
```

## Solution

```c
#include <stdio.h>

#define MAXLINE   1000
#define THRESHOLD 80

int read_line(char s[], int maxsize) {
    int c, i = 0, len = 0;

    while ((c = getchar()) != EOF && c != '\n') {
        ++len;
        if (i < maxsize - 1) s[i++] = c;
    }
    if (c == '\n') {
        ++len;
        if (i < maxsize - 1) s[i++] = '\n';
    }
    s[i] = '\0';
    return len;
}

int main(void) {
    char line[MAXLINE];
    int  len;

    while ((len = read_line(line, MAXLINE)) > 0) {
        if (len > THRESHOLD) {
            printf("[%d] %s", len, line);
            /* if a line longer than MAXLINE was truncated, the buffer
               has no terminating newline - add one to keep output tidy */
            if (line[0] != '\0' && line[/* find last */ 0] != '\n')
                ; /* simpler: just emit newline unconditionally below */
            if (len >= MAXLINE) putchar('\n');
        }
    }
    return 0;
}
```

## Notes from the author

- The threshold is parameterised at the top as `#define THRESHOLD 80`. Even at this scale, putting magic numbers behind a name pays off - and if you ever want to make it a command-line argument, the rename is one place.
- The `read_line` here is exactly the version from ex-1-16, so the bug from §1.9 (under-reporting long-line lengths) is gone. Always reuse the improved version once you have it.
- The `[%d] %s` prefix is mine - the literal exercise just says "print" the line. The length tag makes the output much more useful when you're scanning a file looking for the worst offenders.
- This is a one-line task in modern Unix: `awk 'length > 80' file`. The whole thing is meant to make you appreciate what a "build everything from scratch" environment looks like.
