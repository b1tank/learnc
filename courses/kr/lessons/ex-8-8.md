> Write `bfree(p, n)` which adds an arbitrary block of `n` bytes (e.g. a static buffer) to the free list. After `bfree`, `malloc` can hand out chunks from it. This lets you bootstrap an allocator without calling `sbrk` at all.

## Approach

`bfree` takes a raw memory region and pretends it was already a `malloc`'d block, then `free`'s it. The trick: write a fake header at the start of the region with the size, then call `free` on the byte right after the header.

```c
void bfree(char *p, int n) {
    /* round n down to a multiple of header size */
    n = n / sizeof(Header);
    if (n < 2) return;
    Header *h = (Header *)p;
    h->s.size = n;
    my_free((void *)(h + 1));    /* let free do the insertion + coalesce */
}
```

```c:starter
#include <stdio.h>

typedef long Align;

union header {
    struct {
        union header *ptr;
        unsigned      size;
    } s;
    Align x;
};
typedef union header Header;

static Header  base;
static Header *freep = NULL;

extern void  my_free(void *ap);
extern void *my_malloc(unsigned nbytes);

/* (full malloc/free from §8.7 — abbreviated here) */
void my_free(void *ap) {
    Header *bp = (Header *)ap - 1;
    Header *p;
    for (p = freep; !(bp > p && bp < p->s.ptr); p = p->s.ptr)
        if (p >= p->s.ptr && (bp > p || bp < p->s.ptr))
            break;
    if (bp + bp->s.size == p->s.ptr) {
        bp->s.size += p->s.ptr->s.size;
        bp->s.ptr   = p->s.ptr->s.ptr;
    } else {
        bp->s.ptr = p->s.ptr;
    }
    if (p + p->s.size == bp) {
        p->s.size += bp->s.size;
        p->s.ptr   = bp->s.ptr;
    } else {
        p->s.ptr = bp;
    }
    freep = p;
}

void *my_malloc(unsigned nbytes) {
    unsigned nunits = (nbytes + sizeof(Header) - 1) / sizeof(Header) + 1;
    if (!freep) { base.s.ptr = freep = &base; base.s.size = 0; }
    Header *prevp = freep;
    for (Header *p = prevp->s.ptr; ; prevp = p, p = p->s.ptr) {
        if (p->s.size >= nunits) {
            if (p->s.size == nunits) prevp->s.ptr = p->s.ptr;
            else {
                p->s.size -= nunits;
                p        += p->s.size;
                p->s.size = nunits;
            }
            freep = prevp;
            return (void *)(p + 1);
        }
        if (p == freep) return NULL;     /* no sbrk in this version */
    }
}

void bfree(char *p, unsigned n) {
    unsigned units = n / sizeof(Header);
    if (units < 2) return;
    Header *h = (Header *)p;
    h->s.size = units;
    my_free((void *)(h + 1));
}

/* a static heap to donate */
static char heap[8192];

int main(void) {
    bfree(heap, sizeof heap);          /* donate the whole array */

    char *a = my_malloc(100);
    char *b = my_malloc(200);
    char *c = my_malloc(50);
    printf("a=%p, b=%p, c=%p\n", (void *)a, (void *)b, (void *)c);
    /* all three should be inside the heap[] array */
    printf("inside? a=%d b=%d c=%d\n",
        a >= heap && a < heap + sizeof heap,
        b >= heap && b < heap + sizeof heap,
        c >= heap && c < heap + sizeof heap);

    my_free(b);
    my_free(a);
    my_free(c);
    return 0;
}
```

```output
a=0x... b=0x... c=0x...
inside? a=1 b=1 c=1
```

## Why this is useful

- **Embedded systems** without `sbrk`/`mmap`: the heap is a fixed buffer at link time.
- **Per-purpose arenas**: each thread donates a slab and allocates locally.
- **Testing**: deterministic heaps for unit tests.
- **Bootstrap**: the allocator works without any OS support.

## Why the alignment dance

`Header` is `union {struct {...}; long;}` — typically 16 bytes on 64-bit. `bfree` rounds `n` down to whole units so the resulting block is `Header`-sized. The cast `(Header *)p` requires `p` to be `Header`-aligned; static `char heap[8192]` is *not* guaranteed to be aligned. In production, declare `static Header heap[1024];` so the array is `Header`-aligned.

## Try it

1. Donate two separate non-adjacent buffers via `bfree` twice. Verify `malloc` can use both.
2. After donating, allocate until you get NULL. How much did the donated buffer actually serve, vs. the original size?
3. Use a `_Alignas(Header)` array to fix the alignment portably.

## Notes from the author

- `bfree` is the heart of "bring your own memory" allocators — embedded RTOSes, custom arenas, game engines. The same `free`-coalesce code works regardless of where the memory came from.
- The "donate a buffer" pattern is a beautiful inversion: instead of asking the OS for memory, you tell the allocator "here, use this". One function unifies "extend the heap" and "I had some spare memory lying around".
- Modern allocators have `aligned_alloc`, `posix_memalign`, `_Alignas` — proper tools for alignment. K&R-era code used hand-checked alignment via the `union` trick.
