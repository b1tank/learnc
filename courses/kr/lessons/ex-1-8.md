> **Exercise 1-8.** Read input until EOF and report the number of spaces, tabs, and newlines you saw.

## Approach

The skeleton is the §1.5 read-character loop. Add three counters and three `if` checks inside the body - one per character class you care about.

```c:starter
#include <stdio.h>

int main(void) {
    /* TODO: count blanks, tabs, newlines */
    return 0;
}
```

## Solution

```c
#include <stdio.h>

int main(void) {
    int c;
    int blanks = 0, tabs = 0, newlines = 0;

    while ((c = getchar()) != EOF) {
        if (c == ' ')  ++blanks;
        if (c == '\t') ++tabs;
        if (c == '\n') ++newlines;
    }

    printf("blanks:   %d\n", blanks);
    printf("tabs:     %d\n", tabs);
    printf("newlines: %d\n", newlines);
    return 0;
}
```

```output
blanks:   8
tabs:     1
newlines: 3
```

For an input of three lines containing some spaces and one tab.

## Notes from the author

- Three independent `if`s, not `if`/`else if`, because the conditions are mutually exclusive *anyway* (a character can't be both space and tab). With `else if` you'd avoid one redundant comparison per character - micro-optimisation, but a habit you'll see in tight code.
- A cleaner alternative is a `switch` (preview §3.4): `switch (c) { case ' ': ++blanks; break; case '\t': ++tabs; break; case '\n': ++newlines; break; }`. Slightly more lines, but reads as a small table.
- Consider what counts as "whitespace": C also has `\r` (carriage return), `\v` (vertical tab), `\f` (form feed). The library function `isspace(c)` from `<ctype.h>` returns true for all of them. If you wanted to count *all* whitespace as one bucket, that's the right tool - but be aware it's locale-aware (in some locales the no-break space `0xA0` is also whitespace).
