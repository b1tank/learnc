This site is a learning artifact written while I read K&R cover-to-cover. Each section of the book has a short rewrite in my own words, runnable starter code, and a "try it" list. The exercises are enumerated for every chapter — most are still stubs waiting for a walkthrough.

No signup. No tracking. No JavaScript framework. Plain HTML, one markdown file per lesson, and a C compiler that runs in your browser. Prose is hand-revised against K&R; AI-assisted drafts are verified and rewritten before they land.

Status: early. Chapter 1 is in progress; the rest is scaffolded as stubs. The in-browser C compiler is wired up — the first run on each visit downloads about 10 MB of Runno/clang WebAssembly, then caches it. Contribute a lesson.
