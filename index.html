A hand-written, runnable companion to The C Programming Language (K&R, 2e). Every example runs in your browser.

This site is a learning artifact written while I read K&R cover-to-cover. Each section of the book has a short rewrite in my own words, runnable starter code, and a "try it" list. The exercises are enumerated for every chapter — most are still stubs waiting for a walkthrough.

No signup. No tracking. No JavaScript framework. No AI-generated prose. Plain HTML, one markdown file per lesson, and a C compiler that runs in your browser.

Status: early. Chapter 1 is in progress; the rest is scaffolded. The WebAssembly C compiler is the active milestone — until it lands, the Run button shows a stub message. Contribute a lesson.
